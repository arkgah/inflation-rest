# inflation-rest
A spring boot REST web service for calculating CPI

## Spring boot REST веб-сервис для расчёта индекса инфляции (CPI)

Используется:
* Web
* Data JPA
* Validation
* PostgreSQL
* Security
* JWT
* Test

Для работы с сервисом прилагаются коллекция запросов и переменные окружения для Postman
